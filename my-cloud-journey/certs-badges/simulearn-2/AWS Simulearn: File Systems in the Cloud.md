## Topics Covered

- Shared cloud file storage using Amazon EFS
- Mounting file systems to EC2 instances across Availability Zones
- Storage scalability, elasticity, and high availability in the cloud
- Security controls: VPC security groups, IAM policies, and data encryption

## Key Takeaways

- Learned how to set up and configure a fully managed shared file system in the cloud.
- Understood how multiple EC2 instances can access the same file system simultaneously.
- Gained hands-on experience securing file storage using IAM and VPC security groups.
- Explored how EFS automatically scales storage without manual provisioning.
- Built practical knowledge of cloud storage options and when to use EFS vs S3 vs EBS.

## Resources

- Amazon EFS — set up and configured a shared elastic file system in the cloud
- Amazon EC2 — connected instances to the EFS file system to read and write data
- AWS Systems Manager (SSM) Console — used to manage and interact with EC2 instances without SSH
- Amazon FSx — reviewed as an alternative managed file system option for Windows and high-performance workloads
